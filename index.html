Project 1

Tick tac toe

A simple Tic-Tac-Toe game developed using HTML, CSS, and JavaScript allows two players to take turns marking X's and O's on a 3x3 grid. The game checks for a winner or a tie after each move, displaying the result dynamically. HTML is used for the structure, CSS for styling, and JavaScript for game logic. It offers an interactive and responsive gameplay experience.

Rock Paper Scissors

Rock Paper Scissors is a fun and simple web-based game where you compete against the computer in the ultimate battle of wits and luck! Choose your move Rock, Paper, or Scissors and watch as the computer reveals its choice. The game follows the classic rules: Rock beats Scissors, Scissors beats Paper, and Paper beats Rock. Designed with an interactive interface, the game is a quick and entertaining way to test your reflexes and strategy!
